# ElektroJAM

A project aiming to record and facilitate electronic music jam sessions

Tech:

- MIDI in Chrome (select and connect instrument)
- Talk to DAW from any browser (name your tracks & clips, change some params)
- JS + Canvas (edit midi data, keep original, view timeline, multitrack view, change tempo)
- Audio (save streams from connected soundcard to disk, smart-filename, multitrack or mixdown, possibly stream compressed mix/track)
- Nodejs, Python, Polymer
- 0MQ

pre alpha testing!
